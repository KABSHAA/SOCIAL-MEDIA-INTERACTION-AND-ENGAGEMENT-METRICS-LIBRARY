## 📌 Description
This project is a custom C++ library to track social media engagement using likes, comments, and shares. It demonstrates function overloading and operator overloading.

## ⚙️ Features
- Engagement score calculation
- Post comparison
- Combine posts
- Boost engagement
- Categorize posts (Normal / Trending / Viral)

## 🔁 Function Overloading
- update()
- update(int, int, int)
- update(string)

## ➕ Operator Overloading
- +, -, *
- ==, <
- <<
